# دعوة زفاف — صفحة موقع جاهزة للنشر على GitHub Pages

هذه صفحة React خفيفة باللغة العربية تعرض ثلاثة مواقع (منزل العروس، منزل العريس، والكنيسة) مع أزرار تفتحها في Waze أو خرائط Google على الهاتف.

كيفية الاستخدام
- افتح الملف `index.html` (ينشر على GitHub Pages في جذر الفرع `gh-pages` أو `main`).
- لتحديد المواقع استخدم معلمات الاستعلام في عنوان الـ URL:

  - `bride_lat` و`bride_lng` — إحداثيات منزل العروس
  - `groom_lat` و`groom_lng` — إحداثيات منزل العريس
  - `church_lat` و`church_lng` — إحداثيات الكنيسة

مثال:

`https://<your-username>.github.io/<repo>/?bride_lat=31.778&bride_lng=35.235&groom_lat=32.085&groom_lng=34.781&church_lat=31.771959&church_lng=35.217018`

إضافة صورة للدعوة (اختياري)
- لإظهار صورة الدعوة، أضف المعامل `invite_img` مع رابط للصورة:

`...?invite_img=https://example.com/invite.jpg`

نشر على GitHub Pages
1. أنشئ مستودع وادفع هذه الملفات إلى الفرع `main`.
2. ادخل إعدادات المستودع → Pages → اختر فرع `main` و`/ (root)` ثم احفظ.
3. ستصبح الصفحة متاحة على `https://<your-username>.github.io/<repo>/`.

تعديل لاحق
- يمكنك تحديث الإحداثيات مباشرة في الـ URL كما تريد؛ وسيعرض الزر فتح واز/خرائط التطبيقات على الهاتف المحمول.
